## 1️⃣ **Sidebar احترافي**

### الشكل الجديد:
```
┌────────────────────────┐
│                        │
│  🏠 نظرة عامة          │ ← ذهبي لما تحوم
│                        │
│  👥 الموظفون           │
│                        │
│  ✅ المهام             │
│                        │
│  ⚙️ المستخدمين         │
│                        │
│  📨 الطلبات  [5]       │ ← Badge أحمر متحرك
│                        │
│  📊 التقارير           │
│                        │
│  🕐 سجل التحركات        │
│                        │
│  🔐 النسخ الاحتياطي     │
│                        │
└────────────────────────┘
```

### الألوان:
- **الخلفية:** Dark gradient (أسود → رمادي غامق)
- **الحدود:** Golden border (خط ذهبي رفيع)
- **الأيقونات:** رمادي → ذهبي عند التحويم
- **النشط:** Background ذهبي شفاف + خط ذهبي على اليمين

### التأثيرات:
- ✨ **Hover:** 
  - يتحرك لليسار قليلاً
  - اللون يتحول لذهبي
  - الأيقونة تكبر
  - Background gradient ذهبي شفاف

- 🎯 **Active:**
  - Background ذهبي واضح
  - خط ذهبي على اليمين
  - الأيقونة بـ glow ذهبي
  - Shadow ذهبي خفيف

- 🔴 **Badge:**
  - أحمر gradient
  - يتحرك (pulse animation)
  - Shadow أحمر

## 4️⃣ **الانيميشنات**

### 1. Fade In (عند فتح التاب):
```css
من: opacity 0, translateY(20px)
إلى: opacity 1, translateY(0)
المدة: 0.5s
```

### 2. Hover على Sidebar:
```css
يتحرك: translateX(-5px)
يتلون: golden
الأيقونة: scale(1.1)
```

### 3. Pulse على Badge:
```css
يكبر: scale(1) → scale(1.05)
يصغر: scale(1.05) → scale(1)
يتكرر: كل 2 ثانية
```

### 4. Glow على الأيقونات النشطة:
```css
drop-shadow: 0 0 8px rgba(212,175,55,0.5)
```
